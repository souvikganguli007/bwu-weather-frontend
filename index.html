About the Project

Built for the
Brainware Community

A student-developed weather intelligence platform serving Brainware University in Barasat, West Bengal — combining live API data with academic research.

🔍
Real-Time Search
Fetch live weather for any city worldwide using OpenWeatherMap API with instant results.
🩺
Health Alerts
Monitor AQI, UV index, PM2.5 and CO levels for a personalised campus safety score.
📊
Data Analytics
Historical data stored in MongoDB and visualised with Chart.js for climate trend analysis.
🎓
Academic Use
Used for environmental studies, campus safety alerts and student research projects at BWU.
